Intro
구조물의 완공 또는 시공 과정에서 지하수에 의한 부력(양압력) 에대한 검토가 미비할 경우 구조물의 피해를 초래하는 경우가 많습니다.
따라서, 지중구조물에서 부력에 대한 안정성을 확보하는 것은 필수적입니다.
지반 중 또는 지반과 구조물 사이에 간극수가 존재하는 구조물의 저면에 작용하는 상향의 정수압[부력:(U)]에 의해 생기는 힘과 저항력[구조물 무게:(W1),  상재토압:(W2), 마찰력:(Ps)]에 대한 검토를
수행함으로서 안정성을 확보한 구조물을 설계하여야 합니다.

Version

Language
영어


Benefits of this plugin

구조물을 Midas Civil NX 모델을 기반으로 기본적으로 모든 모델을 불러옴으로써 자동으로 구조물의 자중을 산정합니다.
추가적으로 검토할 구조물 정보를 사용자가 직접 저장함으로서 시공중 부력검토를 수행할 수 있습니다.

부력에 대한 다양한 저항조건을 고려합니다.
* 전단키
* 전단마찰력
* 시공중 검토 (상부토압 무시)
* 추가 상재하중 조건

안정성 검토 결과는 변수와 조건에 따라 실시간으로 확인함으로서 편리하게 부력검토를 수행합니다.
개략적인 검토를 수행할 수 있으며, 부력계산서와 비교함으로서 오류를 방지할 수 있습니다.

How to use this plugin?





유의사항
본 부력검토는 모델링 기반으로 노드점에서 곂쳐지는 요소와 헌치등에 의해 구조물의 규모에 따라 오차가 있습니다. 

Conclusion
모델링 기반으로 빠르게 검토함으로서 전통적으로 수계산하던 방식보다 빠르고 편리하게 부력안정성 검토가 가능합니다.
